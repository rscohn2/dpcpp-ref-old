nd_item¶

template <int dimensions = 1>
class nd_item;

The nd_item describes the location of a point in an nd_range.

An nd_item is typically passed to a kernel function in a parallel_for. It addition to containing the id of the work item in the work group and global space, the nd_item also contains the nd_range defining the index space.
